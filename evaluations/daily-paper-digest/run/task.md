## 固定主题
Agent Systems for LLMs

## 任务说明
请围绕主题 “Agent Systems for LLMs” 完成一次文献检索与初步整理，目标不是泛泛介绍主题，而是尽可能给出可用于后续撰写文献综述的资料基础。

## 输出要求
1. 检索该主题下具有代表性的论文
2. 给出候选论文清单
3. 对论文进行初步分类
4. 每篇论文附上简短说明
5. 尽可能提供来源线索（如 arXiv、会议、链接等）

## 期望输出结构
- 论文标题
- 作者 / 年份
- 来源
- 1～3 句简述
- 所属类别

## 评测关注点
- 是否真实找到相关论文
- 来源是否清楚
- 是否能按方向分类
- 输出是否能支撑后续综述写作
