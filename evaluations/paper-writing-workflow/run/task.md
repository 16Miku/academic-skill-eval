# 测试任务

## 基本信息
- skill：paper-writing-workflow
- 类型：写作 / 工作流型
- 测试轮次：第 1 轮
- 测试日期：待填写
- 执行方式：待填写

## 固定主题
Agent Systems for LLMs

## 任务说明
请围绕主题 “Agent Systems for LLMs” 输出一份文献综述初稿，或至少输出一份明显可用于继续扩展成综述正文的写作结果。

## 任务要求
1. 输出中文内容
2. 总长度目标为 800–1200 字
3. 内容尽量覆盖：
   - 研究背景
   - 关键方向划分
   - 各方向代表工作
   - 当前挑战
   - 总结
4. 尽可能给出引用依据或来源线索
5. 输出不能只停留在写作建议，应有实际写作产出

## 可接受结果
- 一篇较完整的综述草稿
- 或：详细大纲 + 部分正文 + 文献线索

## 评测关注点
- 是否真正推进写作
- 是否具有综述结构
- 是否覆盖关键研究方向
- 是否有文献支撑痕迹
- 输出是否可继续加工为正式综述
